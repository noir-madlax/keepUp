# 中国MCN公司在KOL/KOC合作中的广告费收取模式研究

## 摘要

本报告旨在深入探讨中国多频道网络（MCN）公司在连接品牌方与关键意见领袖（KOL）及关键意见消费者（KOC）过程中的广告费收取模式。基于2024-2025年的市场数据与行业报告，本研究确认了MCN行业的收费结构正从单一的“差价模式”向“服务费+绩效佣金”的复合模式转型。研究发现，虽然行业内普遍流传的“10%-40%”服务费比例在大多数基础服务场景下是准确的，但该比例受平台政策（如抖音星图、小红书蒲公英）、合作深度及责任归属的显著影响。特别是在涉及直播带货与效果承诺（ROI）的合作中，综合费率往往突破40%。本报告详细拆解了不同平台（抖音、小红书）的官方收费标准与MCN的隐性溢价逻辑，对比了MCN与传统4A广告公司的商业模式差异，并为品牌方提供了基于成本效益的合作建议。

---

## 1. 引言：中国MCN行业的商业化生态

### 1.1 研究背景与定义
中国MCN（Multi-Channel Network）机构已从早期的内容搬运与简单的网红经纪，演变为连接内容创作者、平台与品牌方的核心枢纽。截至2023年，中国MCN机构注册数量已超过25,400家 [cite: 1]。在KOL/KOC营销生态中，MCN不仅提供达人资源，更承担了内容孵化、流量分发、商业对接及数据分析等多重职能。

### 1.2 平台生态对收费模式的影响
中国的网红营销高度依赖平台生态，主要以抖音（Douyin）和小红书（Xiaohongshu）为代表。这两个平台建立了严格的官方对接平台——巨量星图（Xingtu）和小红书蒲公英（Pgy），强制规范了交易流程与结算体系。
*   **抖音巨量星图**：作为抖音的官方营销服务平台，它不仅是交易通道，更是数据与资金的监管者 [cite: 2, 3]。
*   **小红书蒲公英**：作为小红书优质创作者的商业合作服务平台，它规定了严格的报备与结算机制，对“私单”进行严厉打击 [cite: 4, 5]。

这种“平台中心化”的结构决定了MCN的收费模式必须建立在平台官方规则的基础之上，形成了“平台服务费（硬性成本）+ MCN服务溢价（弹性利润）”的双层收费结构。

## 2. 广告费收取比例深度调研

### 2.1 广告费构成的分层解析
针对“10%-40%”的波动范围，调研发现这一数据主要指代的是**MCN机构向品牌方收取的服务溢价**或**平台扣除的技术服务费**，具体比例因口径不同而异。完整的广告费由以下三部分组成：

1.  **达人净得报价（Net Price）：** 达人实际到手的收入。
2.  **平台技术服务费（Platform Fee）：** 平台强制收取的“过路费”。
3.  **MCN服务费/溢价（Agency Fee/Markup）：** MCN机构的利润与运营成本覆盖。

### 2.2 平台官方收费标准（硬性基准）
平台服务费是MCN定价的底线成本，2024-2025年的最新标准如下：

#### 2.2.1 抖音巨量星图
*   **基础服务费**：对于常规的短视频传播任务，平台通常收取一定比例的服务费。
*   **电商引流差异化收费**：
    *   **抖音小店闭环**：若推广商品链接来自抖音小店，平台服务费较低，通常为**5%**。这是为了鼓励平台内的电商闭环 [cite: 3, 6]。
    *   **第三方外链**：若推广链接指向淘宝、京东等第三方平台，平台服务费高达**20%**。这直接增加了品牌在MCN合作中的基础成本 [cite: 3, 6]。
*   **结算模式**：星图采用“一口价”或“按效果付费（CPM/CPA）”模式，客户冻结金额包含底价、预估播放量费用及服务费 [cite: 7]。

#### 2.2.2 小红书蒲公英
*   **普通模式**：仅提供内容合作服务，平台收取合作金额的**10%**作为服务费（含税费+平台技术服务费） [cite: 4, 8]。
*   **优效模式**：包含内容合作及平台流量助推，服务费比例提升至**20%**。此模式下，平台承诺笔记曝光量高于该博主近期80%的笔记 [cite: 9, 10]。
*   **结算周期**：个人创作者次月10日结算，扣除10%服务费；机构创作者由机构按月结算 [cite: 4]。

### 2.3 MCN机构的收费比例与波动范围验证
调研显示，**10%-40%的波动范围是准确的**，但需区分不同的收费名目：

*   **纯服务费模式（Service Fee Model）：** 当品牌方直接指定达人，仅委托MCN进行执行对接时，MCN通常收取总预算的**10%-15%**作为执行服务费。
*   **全案代理模式（Agency Model）：** 若MCN提供策略策划、脚本撰写及多达人统筹，服务费比例通常在**20%-30%**。
*   **溢价模式（Markup Model）：** 在“刊例价（Rate Card）”体系中，MCN对外报价通常包含达人底价、平台费及机构利润。调研数据显示，头部MCN的刊例价溢价率（Markup）通常在**30%-40%**甚至更高，这部分包含了MCN的运营成本和利润 [cite: 11]。
*   **直播带货佣金（Commission）：** 在直播带货场景下，MCN收取的佣金比例（CPS）通常在**10%-30%**之间，部分高毛利品类（如美妆）可达40%以上 [cite: 11, 12]。

### 2.4 不同规模MCN的收费差异
*   **头部MCN（如无忧传媒、美One）：** 拥有极强的议价能力和独家头部KOL资源。其收费通常不以简单的“比例”计算，而是采用**高额坑位费+高佣金**模式。由于掌握稀缺流量，其综合溢价率往往超过40% [cite: 13, 14]。
*   **腰部/垂直MCN：** 竞争最激烈，通常严格遵守行业标准的**15%-25%**服务费比例，依靠服务质量和垂类专业度（如美妆、母婴）获客 [cite: 15]。
*   **尾部/小型MCN：** 为争夺客户，往往采用**低服务费（5%-10%）**甚至**免服务费**（仅赚取平台返点或达人差价）的策略，但服务质量和抗风险能力较弱。

## 3. 服务内容与收费关系的深度关联

收费的高低与服务内容的深度、广度及复杂性呈显著的正相关关系。

### 3.1 基础服务（对应10%-15%收费比例）
此类服务通常被称为“执行单”或“走单”，MCN仅作为交易通道。
*   **达人建联与档期预约**：确认达人是否有档期。
*   **合同与财务流程**：处理平台下单、发票开具、税务合规。
*   **基础内容审核**：确保内容不违规，符合品牌基本Brief。
*   **按时发布**：监控达人是否按时发布内容。

### 3.2 标准服务（对应20%-30%收费比例）
这是目前市场上最主流的合作模式，MCN提供增值服务。
*   **达人搜索与筛选**：基于数据工具（如千瓜数据、巨量算数）进行粉丝画像匹配，提供达人List供品牌选择 [cite: 5, 9]。
*   **内容创作优化**：MCN的编导团队介入，协助达人打磨脚本，优化分镜，确保品牌信息植入自然且具有传播力 [cite: 13]。
*   **投放效果监测**：实时监控点赞、评论、转发数据，处理异常评论。
*   **数据分析报告**：结案时提供包含CPE（单次互动成本）、CPM（千次曝光成本）等维度的详细复盘报告 [cite: 16]。

### 3.3 深度/全案服务（对应30%-40%+收费比例）
此类服务接近4A广告公司的标准，MCN深度介入品牌营销策略。
*   **全案策略制定**：制定阶段性的种草策略（如“金字塔模型”投放：1个头部+N个腰部+海量KOC） [cite: 9]。
*   **KOC矩阵铺量**：组织数百甚至上千名KOC进行规模化刷屏/晒单，涉及复杂的统筹管理。
*   **投流优化（SEM/Feed）**：结合信息流广告（如抖音Dou+、小红书聚光）进行赛马机制投放，放大优质内容流量 [cite: 17]。
*   **舆情管理与危机公关**：在评论区进行舆论引导，防范负面信息扩散。

### 3.4 收费与广告金额的关系
*   **规模效应**：通常整体广告金额越大，MCN愿意给予的服务费折扣越高（例如从20%降至15%），但绝对收益额增加。
*   **返点机制（Rebate）**：在巨额投放中（如百万级以上），MCN可能会向品牌方提供“返点”，这在行业内是公开的秘密。返点通常来自于平台给MCN的年度激励，MCN将其部分让利给品牌以换取大额预算 [cite: 18]。

## 4. 责任与收费比例的博弈：合作模式研究

品牌方对结果确定性的追求推动了收费模式的演变，责任越重，收费越高。

### 4.1 常见合作模式与定价

#### 4.1.1 CPT (Cost Per Time) - 一口价模式
*   **定义**：按单条视频或单场直播付费，不承诺转化效果。
*   **收费特征**：标准刊例价，风险主要由品牌方承担。
*   **适用场景**：品牌曝光、新品上市宣发、头部达人合作。
*   **收费比例**：标准服务费率（10%-20%）。

#### 4.1.2 CPS (Cost Per Sales) - 佣金模式
*   **定义**：按实际销售额支付佣金。
*   **收费特征**：低坑位费（或纯佣金）+ 高比例佣金（20%-40%）。
*   **适用场景**：直播带货、挂车短视频。
*   **责任关系**：MCN承担销量风险，因此要求更高的利润分成上限 [cite: 12, 14]。

#### 4.1.3 VAM (Valuation Adjustment Mechanism) - 对赌模式
*   **定义**：MCN承诺达到特定的ROI（投入产出比）或曝光量（如保底100万播放）。
*   **收费特征**：**高溢价**。若MCN承诺保量，通常会在基础报价上增加**20%-50%**的风险溢价，或者要求更高比例的超额奖励。
*   **案例**：小红书的“优效模式”即为平台级的轻度对赌（多收10%服务费，保曝光量） [cite: 9]。
*   **风险**：若未达标，MCN需进行补量或退款；若超标，MCN可能要求超额分红。

### 4.2 品牌方如何选择
*   **追求声量（Brand Awareness）**：选择CPT模式，重点考核MCN的内容创意能力和账号筛选精准度，支付标准服务费。
*   **追求销量（Sales Conversion）**：选择CPS或“坑位费+CPS”模式，重点考核MCN的选品匹配度和过往带货数据，接受较高的佣金比例以换取确定性 [cite: 14]。
*   **预算有限的新品牌**：建议选择KOC铺量模式，通过MCN打包采购（Package Deal），降低单次合作的边际成本。

## 5. MCN公司 vs 广告公司（4A）：对比分析

随着MCN能力的提升和广告公司业务的下沉，两者界限日益模糊，但在本质基因和收费逻辑上仍有显著差异。

### 5.1 本质区别与优劣势

| 维度 | MCN公司 (Multi-Channel Network) | 广告公司 (4A/Creative Agency) |
| :--- | :--- | :--- |
| **核心资产** | **人（Influencer/IP）**：拥有达人的签约权和管控力 [cite: 19]。 | **策略（Strategy/Idea）**：拥有品牌方法论和创意体系 [cite: 20]。 |
| **创意逻辑** | **自下而上**：基于达人的人设和粉丝喜好产出内容，原生感强，但不可控性高。 | **自上而下**：基于品牌核心信息（Key Message）拆解，统一性强，但可能缺乏网感 [cite: 20]。 |
| **响应速度** | **极快**：适应短视频和直播的快节奏，能迅速追热点。 | **较慢**：流程繁琐，层层汇报，制作周期长。 |
| **优势** | 执行力强、成本相对低、流量获取直接、达人配合度高。 | 品牌调性把控严谨、策略完整、跨媒体整合能力强。 |
| **劣势** | 策略深度不足、品牌理解可能偏差、数据造假风险。 | 价格昂贵、对新媒体语境反应滞后、KOL采购成本高（中间商多）。 |

### 5.2 收费模式对比

*   **MCN模式：以资源为导向的差价/佣金制**
    *   主要赚取达人报价的差价（Markup）或平台返点。
    *   收费项通常打包在“KOL投放费用”中，较少单独列支巨额的“创意费”或“策略费”。
    *   **典型报价单**：达人A（3万）+ 达人B（5万）+ ... + 服务费（总价的10%）。

*   **广告公司模式：以人时为导向的月费/项目制**
    *   主要收取**Retainer Fee（月费）**或**Project Fee（项目费）**。
    *   人员工时（Hourly Rate）是核心计费依据。
    *   KOL采购通常作为第三方成本（Third Party Cost），广告公司会加收**15%的代理费（Agency Fee）**，但其核心利润来自于策略和创意产出，而非仅仅是媒介采买 [cite: 21]。

### 5.3 品牌选择建议
*   **选择MCN的情况**：
    *   需要快速铺量、制造热点（如新品上市种草）。
    *   以销售转化为直接目标（直播带货）。
    *   预算主要用于媒介投放，对TVC级大创意需求不高。
    *   需要深度绑定某位头部达人（直接找其所属MCN最划算）。
*   **选择广告公司的情况**：
    *   品牌重塑或大型年度战役（Campaign），需要统一的大策略。
    *   跨媒介整合（不仅是抖音/小红书，还涉及线下、OTV等）。
    *   对品牌安全（Brand Safety）和调性有极高要求（如奢侈品）。

## 6. 市场调研数据与行业标准（2024-2025）

### 6.1 典型收费案例
*   **小红书图文种草案例**：
    *   **博主层级**：腰部博主（5w-50w粉丝）。
    *   **报价区间**：3,000 - 8,000元/篇 [cite: 4]。
    *   **MCN收费**：若品牌委托MCN采购10位此类博主，总预算约5万元。MCN会收取约**5,000-10,000元**的服务费（10%-20%），或直接打包报价6万元（含服务费）。
*   **抖音直播带货案例**：
    *   **合作模式**：坑位费 + 佣金。
    *   **费用结构**：坑位费覆盖达人开播成本（几千至几万不等），佣金比例**20%**（其中5%给平台，15%由MCN和达人分） [cite: 3]。
    *   **MCN收益**：主要依赖GMV（交易总额）抽成。

### 6.2 行业趋势
*   **去中介化与再中介化**：虽然平台试图让品牌直接对接达人（去中介），但由于达人管理难度大，品牌依然倾向于通过专业MCN（再中介）进行规模化管理，但对MCN的“服务价值”要求更高，纯倒手的MCN生存空间被压缩 [cite: 1, 22]。
*   **税务合规化**：随着国家对网红税务的严查，MCN在报价时会更严格地列明税费（通常为6%或更高），这部分成本正逐渐转嫁给品牌方 [cite: 23]。

## 7. 对品牌方的建议

基于上述调研，为品牌方提供以下策略建议：

1.  **厘清“含税/含费”报价**：在询价时，务必明确MCN的报价是否包含平台服务费（星图/蒲公英费用）及税点。通常MCN报出的“刊例价”是含税含费的，但“净价”则不含。
2.  **善用平台工具核价**：品牌方应注册巨量星图或小红书蒲公英的企业账号，直接查看达人的公开报价。将此价格与MCN的打包报价对比，计算溢价率。若溢价率超过20%且无额外深度服务，应进行议价 [cite: 24, 25]。
3.  **签署“对赌”或“保量”条款**：对于预算较大的投放（如超过10万元），建议与MCN签署KPI考核条款（如保底阅读量、互动率），将部分服务费挂钩效果，降低沉没成本。
4.  **组合拳策略**：
    *   **策略层**：聘请广告公司或咨询公司做顶层设计。
    *   **执行层**：直接对接垂直领域的MCN进行采购，避免广告公司再转包给MCN造成的“双重溢价”。
5.  **关注数据真实性**：MCN为了交付效果，存在刷量行为。品牌方需关注“水下数据”，利用第三方监测工具（如秒针、克劳锐）验证粉丝画像和互动真实性，拒绝为虚假繁荣付费 [cite: 26]。

## 结论

中国MCN公司的广告费收取模式已进入精细化运营阶段。**10%-40%**的费用波动真实反映了从“基础通道服务”到“深度策略服务”的价值阶梯。品牌方在合作中应充分理解平台规则（如抖音星图20%的第三方链接费、小红书10%的服务费），并根据自身的营销目标（声量vs销量）选择匹配的计费模式（CPT vs CPS）。未来，具备全案策划能力、能对效果负责的MCN将获得更高的溢价权，而仅靠信息差赚取差价的MCN将被市场边缘化。
